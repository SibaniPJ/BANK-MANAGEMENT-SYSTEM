# BANK-MANAGEMENT-SYSTEM
A Java-based Bank Management System designed to handle essential banking operations such as account creation, transactions, and customer record management. This project provides a user-friendly interface for both administrators and customers to perform various banking activities efficiently.

## Project Overview
A simple Bank Management System built using Java Swing for the GUI.
This application allows users to manage bank accounts, perform transactions, and view account details in a user-friendly interface.

##Features
Create new bank accounts
View account details
Deposit and withdraw money
Transfer funds between accounts
Delete accounts
User-friendly GUI using Java Swing

## Technologies Used
Java SE
Java Swing for GUI
IntelliJ IDEA as the IDE
MySQL for storing account data
